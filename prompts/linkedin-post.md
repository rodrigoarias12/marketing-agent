# LinkedIn Post Template

## Format Rules

- 150-300 words
- Start with a hook: bold statement, surprising data, or direct question
- Use line breaks for readability. No walls of text.
- End with a question or CTA to drive engagement
- Include 3-5 relevant hashtags at the bottom (separated from body)

## Post Types

Use one of these types per post:

- **insight_post**: Share a data point or observation with personal analysis
- **hot_take**: Controversial but constructive opinion on an industry trend
- **building_update**: Share progress on what you're building (building in public)
- **metric_share**: Lead with a number, explain why it matters
- **storytelling**: Personal narrative with a lesson or takeaway

## Structure

Each post MUST use this exact markdown format for the dashboard parser:

```

━━━ POST ━━━━━━━━━━━━━━━━━━━━━━━━━
PLATFORM: LinkedIn
TYPE: <one of the types above>
PILLAR: <thematic pillar>
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

<post body here>

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CTA: <call to action description>
HASHTAGS: <#Tag1 #Tag2 #Tag3>
```

## Quality Check

Before finalizing, verify:
- [ ] Hook grabs attention in first line
- [ ] Body has a clear point (not rambling)
- [ ] Ends with engagement driver (question or CTA)
- [ ] Hashtags are relevant, not generic
- [ ] Tone matches base-voice.md
