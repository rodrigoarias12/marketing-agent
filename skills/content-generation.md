```

[1-2 líneas de contexto o insight]

[CTA sutil: pregunta, follow, link]
```

## TikTok/Reels Templates

### "60 Second Fintech"
```
HOOK (0-3s): "Sabías que [dato sorprendente]?"
TEXTO EN PANTALLA: Pregunta o afirmación bold
EXPLICACIÓN (3-45s): Concepto simple con visuales
TIE-IN (45-55s): "Eso es exactamente lo que resolvemos en [YourBrand]"
CTA (55-60s): "Link en bio"
```

### Behind the Scenes
```
HOOK (0-3s): "Así se ve construir una fintech en LATAM..."
ESCENA 1 (3-15s): Workspace / código
ESCENA 2 (15-35s): El desafío del día
ESCENA 3 (35-50s): Cómo lo resolvimos
CTA (50-60s): "Follow para más behind the scenes"
```

### Screen Recording Tech
```
HOOK (0-3s): "La tech detrás de [X]..."
SCREEN (3-45s): Code walkthrough con voiceover
WHY (45-55s): "Elegimos esto porque..."
CTA (55-60s): "Tu stack? Comentá 👇"
```

## YouTube Templates

### Tech Deep Dive (15-20 min)
```
0:00 - Hook: mostrar resultado final
0:30 - Contexto: por qué lo construimos
2:30 - Arquitectura: diseño del sistema
5:30 - Implementación: code walkthrough
13:30 - Resultados: métricas
15:30 - Lecciones: qué haríamos diferente
17:30 - CTA: suscribirse, yoursite.com
```

### Founder Vlog (10 min)
```
0:00 - Hook: momento más interesante de la semana
0:15 - Intro: "Semana [N] de building [YourBrand]..."
0:45 - Updates: qué se shipeó
5:45 - Desafío: problema principal y resolución
8:45 - Métricas
10:00 - CTA
```
