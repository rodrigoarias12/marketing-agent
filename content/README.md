# Content Directory

This folder contains content generated by Eddie, your AI marketing agent.

## Structure

```
content/
  drafts/          # Generated content drafts (one file per day)
  images/          # Generated images (HTML sources + PNGs, organized by date)
  published/       # Log of published content with performance data
```

## Getting Started

1. Configure your brand in `SOUL.md` and `USER.md`
2. Set up your API keys (see `docs/API-SETUP-GUIDE.md`)
3. Run Eddie to generate your first content batch
4. Drafts will appear in `content/drafts/YYYY-MM-DD.md`

See `content/drafts/EXAMPLE-2026-01-01.md` for an example of the draft format.

## Notes

- Real draft files are gitignored (they contain your specific content)
- Only `EXAMPLE-*.md` files are tracked in git
- Images are fully gitignored (generated per user)
- Published logs are gitignored (contain your posting history)
